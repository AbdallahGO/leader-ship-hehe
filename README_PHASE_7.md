## 🚀 How to Activate

### Quick Start (5 minutes)
```bash
# Navigate to backend
cd backend

# Install & configure
composer install
cp .env.example .env
php artisan key:generate

# Run migrations & seeders
php artisan migrate
php artisan db:seed

# Run tests
php artisan test

# Start server
php artisan serve
```

Visit: `http://localhost:8000/api/v1/admin/users`
