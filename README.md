# Apex Leadership Summit

Static HTML/CSS/JS site for the Apex Leadership Summit 2026.

## Development

1. Start the backend:
   ```bash
   cd apex-speckit/backend
   npm install
   npm run migrate
   npm run dev
   ```

2. Start the frontend:
   ```bash
   npm install
   npm run serve
   ```
   Then open http://localhost:8000/leadership-summit.html

## Production

Deploy to GitHub Pages from the repository root.

Backend deployed to Railway at https://api.apexsummit.org
